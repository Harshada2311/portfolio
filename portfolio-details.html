Building Web APIs with ASP.NET Core 8

Modern web applications provide much of their functionality in an API. In this course, learned you how to design and implement such an API using ASP.NET Core 8. Get started with minimal APIs and the basics of HTTP, REST, and API design. Learn how to retrieve and write data, and practice what you learn with hands-on challenges. Explore how to secure your API with authentication and authorization, and how to document it with OpenAPI. Finally, learn how to deploy your API to Azure and monitor its performance.

Learning Angular

Angular is a powerful framework for building web applications. In this course, learned the fundamentals of Angular, including components, templates, and services. Explore how to create a single-page application (SPA) with Angular and how to use Angular CLI to streamline your development process. Learn how to work with forms, handle user input, and manage state in your application. Finally, discover how to deploy your Angular application and make it available to users. This course is perfect for anyone looking to get started with Angular or enhance their existing skills. With hands-on challenges and real-world examples, you'll gain the knowledge and confidence to build your own Angular applications.
